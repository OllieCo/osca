# Dispatcher SLO Definitions

**Version:** v1.0 (2026-04-25)
**Owner:** Platform Engineering
**Review cadence:** Quarterly or after any SLA breach

---

## SLOs

### 1. Availability — API

| Version | Target | Measurement window | Error budget |
|---------|---------|--------------------|-------------|
| v1.0 | 99.5% | 30-day rolling | 3.6 hours/month |
| v2.0+ | 99.9% | 30-day rolling | 43.8 minutes/month |

**Measured by:** Grafana Synthetic Monitoring probes from SYD + SGP to `GET /api/health`
**Error defined as:** HTTP 5xx or probe timeout > 5s

### 2. Latency — Supervision workflow (p95)

| Version | Target | Measurement window |
|---------|--------|--------------------|
| v1.0+ | p95 < 2s | 5-minute rolling |

**Measured by:** Grafana Mimir RED-method histogram on `/api/agent` endpoint
**Excludes:** Cold-start model loads (first request after Ollama restart)

### 3. Latency — Relief allocation workflow (p95)

| Version | Target | Measurement window |
|---------|--------|--------------------|
| v1.0+ | p95 < 5s | 5-minute rolling |

**Measured by:** Grafana Mimir histogram on `/api/chat` endpoint

### 4. TTFT — Time-to-first-token (inference)

| Version | Target | Measurement window |
|---------|--------|--------------------|
| v1.0+ | TTFT < 1s | 5-minute rolling |

**Measured by:** `durationMs` field in `inference complete` Pino log, exported to Mimir

---

## Error budget policy

| Remaining budget | Action |
|-----------------|--------|
| > 50% | Normal release cadence |
| 25–50% | Review open P1/P2 issues; no new risky releases without sign-off |
| < 25% | **Release freeze** — no deploys except hotfixes for the incident causing burn |
| 0% | Post-mortem required before next release; customer comms if SLA breached |

**Release freeze trigger:** Documented in [CI/CD playbook](../server/docs/cicd-playbook.md) (to be created). CI/CD pipeline checks Grafana API for current error-budget status; blocks deploy if < 25% remaining.

---

## SLA (customer-facing commitment)

| Tier | Commitment | Consequence of breach |
|------|------------|----------------------|
| School (paid) | 99.5% monthly availability | Service credit: 1-day extension per hour down (capped at 7 days) |
| Pilot | Best effort | No credit |

**Source of truth for SLA calculations:** Monthly SLO report auto-generated by Grafana (once platform is provisioned — D-34).
